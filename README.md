# P4 — Preditor de Satisfação Multilíngue

NLP que analisa reviews em PT/EN/ES e prevê score de satisfação.

## Stack
- **Frontend:** React (Vite) + TypeScript
- **Backend:** Python Flask + NLP (TF-IDF + Ridge)
- **MCP Server:** Node + Express
- **Banco:** Neon (PostgreSQL)
- **LGPD:** Presidio anonymizer
- **Cron:** GitHub Actions
- **Deploy:** Koyeb
